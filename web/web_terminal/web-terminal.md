# web-terminal

Web Terminal 是一个基于 Web 技术实现的终端界面，允许用户通过浏览器访问和操作远程计算机或服务器上的命令行界面。它为用户提供了一种方便、跨平台、无需安装任何软件的方式来访问和管理远程计算资源。

技术选型：xterm + websocket
